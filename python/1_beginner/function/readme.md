# Function

## Why Use Function?

- **분해(Decomposition)**
- **추상화(Abstraction)**

### Decomposition

- 기능 분해
- 재사용

### Abstraction

- 함수의 내부 원리를 몰라도 괜찮다.
- 함수의 이름, 기능만 알고 활용할 수 있다.
