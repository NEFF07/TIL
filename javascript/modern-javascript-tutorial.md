## 객체

### 점 표기법과 대괄호 표기법

대괄호 표기법은 프로퍼티 key, value에 대한 제약을 없애, 수식이나 할당을 사용할 수 있게 한다.

일반적으로 점 표기법을 쓰되, 필요한 경우에 대괄호 표기법을 사용하자.
